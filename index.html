FAQ

Questions fréquentes

Mon Budget ne se connecte jamais à votre banque. Vous saisissez vous-même vos transactions. Les données sont stockées localement sur votre appareil. En version Premium, elles sont synchronisées de façon chiffrée via Firebase, accessibles uniquement par votre compte.
La version Gratuite inclut 1 an d'historique, 3 catégories de dépenses variables et toutes les fonctionnalités de base. La version Premium ajoute l'historique illimité, la synchronisation cloud en temps réel sur 3 appareils, et les catégories illimitées.
Oui ! Mon Budget est une PWA (Progressive Web App) qui fonctionne entièrement hors ligne. Les données sont sauvegardées localement, puis synchronisées avec le cloud (Premium) dès que vous retrouvez une connexion.
Ouvrez le site dans Chrome (Android) ou Safari (iPhone), puis utilisez le menu "Ajouter à l'écran d'accueil". L'app s'installe comme une application native, sans passer par le Play Store ou l'App Store.
