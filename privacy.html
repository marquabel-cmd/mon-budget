🔐 Politique de confidentialité

Mon Budget — Application de gestion mensuelle des revenus et dépenses

← Retour à l'application

📋 Qui sommes-nous ?

Mon Budget est une application web progressive (PWA) permettant de gérer ses revenus et dépenses mensuels. Elle est développée et maintenue par Pascal Marquabel.

Contact : pascalair@gmail.com

📦 Quelles données collectons-nous ?

Nous collectons uniquement les données strictement nécessaires au fonctionnement de l'application :

  • Adresse e-mail — utilisée pour l'identification de votre compte
  • Données de budget — revenus, charges et dépenses que vous saisissez vous-même
  • Identifiant d'appareil — un code généré localement pour limiter l'utilisation à 3 appareils par compte
  • Date de dernière connexion — pour la gestion des comptes

Nous ne collectons pas : nom complet, numéro de téléphone, données bancaires réelles, données de localisation.

🔒 Comment vos données sont-elles stockées ?

Vos données sont stockées de deux façons :

  • Localement sur votre appareil (localStorage du navigateur) — accessibles uniquement depuis votre appareil
  • Dans le cloud via Firebase (Google), hébergé en Europe (europe-west1) — uniquement si vous avez un compte Premium et que vous êtes connecté

Les données cloud sont protégées par des règles de sécurité Firebase : seul vous pouvez accéder à vos propres données, grâce à votre identifiant unique (UID).

🔑 Connexion via Google ou Facebook

Si vous choisissez de vous connecter via Google ou Facebook, nous recevons uniquement votre adresse e-mail et un identifiant unique fourni par ces services. Nous n'accédons pas à votre profil, vos contacts ou vos publications.

Ces connexions sont gérées par Firebase Authentication (Google). Consultez la politique de confidentialité de Google.

🤝 Partage des données

Nous ne vendons, ne louons et ne partageons vos données avec aucun tiers à des fins commerciales.

Les seuls destinataires de vos données sont :

  • Firebase / Google — pour l'authentification et la synchronisation cloud
  • Vous-même — vous avez un accès complet à toutes vos données

⚖️ Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès — vous pouvez exporter toutes vos données depuis l'application (bouton CSV/JSON)
  • Droit de rectification — vous pouvez modifier vos données directement dans l'application
  • Droit à l'effacement — vous pouvez supprimer votre compte et toutes vos données
  • Droit à la portabilité — vos données sont exportables en JSON

Pour exercer ces droits, contactez-nous à : pascalair@gmail.com

🗑️ Suppression de vos données

Pour demander la suppression de toutes vos données personnelles :

  • Envoyez un e-mail à pascalair@gmail.com avec l'objet "Suppression de données" et votre adresse e-mail de compte
  • Vos données seront supprimées dans un délai de 30 jours

Vous pouvez également effacer vos données locales à tout moment en vidant le cache de votre navigateur.

🍪 Cookies et stockage local

L'application utilise le localStorage de votre navigateur (et non des cookies) pour sauvegarder vos données localement. Ce stockage reste sur votre appareil et n'est pas envoyé à des serveurs tiers.

Aucun cookie publicitaire ou de traçage n'est utilisé.
