# Product Marketing Context — Mon Budget

*Last updated: Mai 2026*

## Product Overview
**One-liner:** L'app de budget familial gratuite qui garde vos données chez vous — sans pub, sans tracking.
**What it does:** Mon Budget est une PWA (Progressive Web App) qui permet de gérer ses revenus, charges fixes et dépenses variables par catégorie. Elle fonctionne hors-ligne, s'installe sur mobile, et importe les relevés bancaires CSV automatiquement. La synchronisation cloud est optionnelle et chiffrée.
**Product category:** Application de gestion de budget personnel / familial
**Product type:** PWA freemium (web + mobile installable)
**Business model:** Gratuit (1 an, catégories limitées) + Premium 2,99€/mois ou 24,99€/an (années illimitées, catégories illimitées, sync cloud multi-appareils)
**URL:** budget.marquabel.be

## Target Audience
**Géographie:** Belgique, France, Suisse, Luxembourg — francophones
**Profils:**
- Familles (25-45 ans) qui veulent contrôler leurs dépenses mensuelles
- Jeunes adultes (18-30 ans) qui commencent à gérer leur budget seuls
- Indépendants qui veulent séparer leurs finances perso
**Primary use case:** Savoir exactement où part l'argent chaque mois
**Jobs to be done:**
- "Je veux voir si je peux me permettre cette dépense"
- "Je veux comprendre pourquoi je suis à découvert en fin de mois"
- "Je veux suivre mes remboursements mutuelle/assurance"
- "Je veux prévoir mon budget pour les fêtes/vacances"

## Problems & Pain Points
**Core problem:** Les gens ne savent pas où part leur argent — fin de mois surprise, découvert inexpliqué.
**Why alternatives fall short:**
- Bankin'/Linxo : aspirent toutes les données bancaires vers leurs serveurs, payants rapidement
- Excel : trop complexe, pas mobile-friendly, pas de calcul automatique
- YNAB : en anglais, trop cher (14$/mois), complexe
- Budgetbakers/Wallet : interface lourde, données dans le cloud obligatoire
**What it costs them:** Stress, découverts, impossibilité d'épargner, conflits de couple sur l'argent
**Emotional tension:** Honte de ne pas "savoir gérer son argent", peur de regarder ses comptes

## Competitive Landscape
**Direct:** Bankin', Linxo — tombent à cause des données obligatoires dans le cloud et du prix
**Secondary:** Excel/Google Sheets — tombent à cause de la complexité et du manque de mobile
**Indirect:** Ne rien faire / regarder son solde bancaire — tombent car aucune vision d'ensemble

## Differentiation
**Key differentiators:**
- Données stockées localement (pas de cloud obligatoire)
- Zéro pub, zéro tracking
- Fonctionne hors-ligne (PWA installable)
- Import CSV des banques belges/françaises
- Gratuit pour commencer, sans carte bancaire
- Interface en français, pensée pour la Belgique
**Pourquoi c'est mieux:** Confidentialité totale + simplicité + gratuité
**Pourquoi les clients choisissent Mon Budget:** "Je veux pas que mon banquier voit que j'utilise une app de budget" / "Je veux pas payer 10€/mois juste pour suivre mon budget"

## Objections
| Objection | Réponse |
|-----------|---------|
| "Et si je perds mon téléphone ?" | La sync cloud Premium protège tout — sinon l'export manuel suffit |
| "C'est sécurisé ?" | Vos données ne quittent jamais votre appareil en version gratuite |
| "C'est compliqué à configurer ?" | Mode Démo disponible, données fictives pour tester sans inscription |
| "2,99€/mois c'est encore cher" | Option annuelle à 24,99€ = 2,08€/mois, moins cher qu'un café |

**Anti-persona:** Quelqu'un qui veut la connexion bancaire automatique (Open Banking) — Mon Budget est manuel/CSV uniquement.
